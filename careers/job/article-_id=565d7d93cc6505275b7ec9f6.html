为什么加入 Teambition 增长团队？

与最优秀的人一起工作： Teambition 增长团队由 CEO 直接领导，他综合能力强，能够帮助你快速学习、创造不同寻常的成就。与 Teambition 研发团队紧密配合，Teambition 的研发团队在国内创业领域享誉盛名，你能遇到 MongoDB 中国社区的主要组织者，AngularJS.cn 站长、 jsGen 与 ThenJS 的开发者。增长团队的产品经理中有创立了上海知名教育创业企业的联合创始人，也有法籍 Dropbox 前员工，与他们一起工作创意横生。增长团队的数据分析师也是沪上知名的数据专家，他会带大家一起分析数据，讨论合适的指标，并通过数据来验证想法，指导行动。和这些人一起，你会快速成长，并为 Teambition 的增长贡献重要力量。

在自由、享受的环境工作: Teambition 的上海办公室是一间近一千平米的圆形开放式办公室，我们的办公桌沿着圆形分布：你可以在自己的工作桌上完成自己的工作，也可以坐在沙发上惬意地开始自己的工作，或是在安静的休息区休息一下，为之后的工作补充能量；我们还提供免费的健康午餐和晚餐；每周都有的 AMA，由 CEO 和 COO 主持的问答活动，你可以向他们大胆地提出质疑，这也是了解公司最新进展最好的方法；每年都有公司提供的体检和洗牙；除此之外，你的团队负责人还享有预算，给你的团队创造独占福利。

用最好用的工具工作： Teambition 信赖工具的力量，我们使用自己开发的团队协作工具和沟通工具推进日常工作；在分析的过程中，我们也会使用更多的商业、开源工具，如：Tableau, Kibana, Segment, Customer.io, Fullstory 等等，借助优秀的工具来提高生产力。Teambition 给所有人配备了电动升降桌，给大家更舒适的工作环境。更多更好的工具，只等你来提出。当工具都已备齐，你就可以开始创造了。

你的工作职责

  • 配合增长产品经理完成增长项目的研发；
  • 配合数据分析师完成相关的部署；
  • 根据增长团队的需求，研发相应的内部工具；
  • 与研发团队协作，完成增长项目的发布；
  • 与设计团队协作，确保增长项目的设计水准。

我们对你的期望

  • 具有计算机或相关专业学士及以上学位，或相等的自学经历；
  • 具有3年以上 C/C++/Objective-C 编程工作经验；
  • 具有3年以上 iOS 客户端开发经验；
  • 了解 Restful API 的设计原理，有基于 Restful API 的开发经验；
  • Teambition 用户优先；
  • 有 Swift 开发经验者优先；
  • 能使用英语作为工作语言者优先；
  • 有独立作品，或 1 个以上成熟项目经验并可以演示者优先；
  • 在 GitHub 上有 200 天以上活跃天数，或有被其他人加星的项目优先；
  • 热爱工作，在某些方面能惊呆小伙伴者优先。
